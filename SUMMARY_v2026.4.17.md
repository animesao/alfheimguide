### 3. ✅ База данных
**Новая таблица: `release_snapshots`**
```sql
CREATE TABLE release_snapshots (
    id INTEGER PRIMARY KEY,
    tracked_user_id INTEGER,
    repo_name VARCHAR(100) NOT NULL,
    release_tag VARCHAR(100) NOT NULL,
    release_name VARCHAR(255),
    published_at DATETIME NOT NULL,
    is_prerelease INTEGER DEFAULT 0,
    is_draft INTEGER DEFAULT 0,
    FOREIGN KEY (tracked_user_id) REFERENCES tracked_users(id)
)
```

**Индексы:**
- `idx_release_snapshots_user_repo` - для быстрого поиска по пользователю и репозиторию
- `idx_release_snapshots_tag` - для поиска по тегу

## 🚀 Как использовать

### 1. Обновить бота
```bash
git pull origin main
pip install -r requirements.txt --upgrade
```

### 2. Мигрировать базу данных
```bash
python migrate_database.py
```

Это создаст таблицу `release_snapshots` и индексы.

### 3. Запустить бота
```bash
python main.py
```

### 4. Добавить отслеживание
```bash
/add_user github-username
```

Теперь бот будет отслеживать:
- 📝 Коммиты и пуши
- 🚀 Новые репозитории
- 🎉 **Релизы** (новое!)
- 🗑️ Удалённые репозитории
