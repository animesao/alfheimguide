# Changelog - Alfheim Guide Bot

## Версия 2026.4.15 (15.04.2026) - MAJOR UPDATE 🎉

### 🆕 Новые системы

#### 💰 Экономическая система
- **Виртуальная валюта** с кошельком и банком
- **Ежедневные награды** `/daily` — получай монеты каждый день
- **Система работы** `/work` — 8 различных работ для заработка
- **Переводы** `/transfer` — отправляй монеты друзьям
- **Банковская система** `/deposit` `/withdraw` — храни монеты в безопасности
- **Магазин** `/shop` — покупай роли и предметы
- **Таблица лидеров** `/leaderboard` — самые богатые участники
- **Управление магазином** `/shop_add` `/shop_remove` — для администраторов
- **История транзакций** — все операции записываются

#### 📊 Система статистики
- **Топ участников** `/topmembers` — самые активные пользователи (день/неделя/месяц/всё время)
- **Статистика каналов** `/channelstats` — активность по каналам за 7 дней
- **Статистика сервера** `/serverstats` — полная аналитика сервера
- **Статистика пользователя** `/userstats` — детальная информация об активности
- **Графики активности** `/activity_graph` — визуализация активности (1-30 дней)
- **Отслеживание сообщений** — автоматический подсчёт всех сообщений

#### 🛡️ Расширенная модерация
- **Логирование сообщений** — отслеживание удалённых и отредактированных сообщений
- **Система жалоб** `/report` — пользователи могут жаловаться на нарушителей
- **Управление жалобами** `/reports` `/report_resolve` — для модераторов
- **Анти-рейд защита** `/raid_protection` — автоматическая защита от массовых входов
- **Анти-спам** — умное обнаружение спама (5+ сообщений за 5 секунд)
- **Массовые действия** `/massban` `/masskick` — бан/кик нескольких пользователей
- **История предупреждений** `/warnings` — просмотр всех варнов пользователя
- **Управление slowmode** `/slowmode` — установка медленного режима
- **Каналы логов** — отдельные каналы для разных типов логов

#### 🔧 Утилиты
- **Система напоминаний** `/remind` — установка напоминаний (10s, 5m, 2h, 1d)
- **Управление напоминаниями** `/reminders` `/reminder_cancel`
- **Интерактивные опросы** `/poll` — создание опросов с реакциями (до 5 вариантов)
- **Результаты опросов** `/poll_results` — просмотр результатов с процентами
- **Информация о сервере** `/serverinfo` — детальная информация
- **Информация о пользователе** `/userinfo` — профиль пользователя

## Версия 2.0.0 (02.04.2026)

### 🆕 Новое

#### Система верификации (полностью переработана)
- **Единое меню настроек** `/verify` с кнопками для каждого раздела
- **Готовые пресеты верификации:**
  - 📋 Простая верификация — одна кнопка, одна роль
  - 📜 С подтверждением правил — сначала правила, потом согласие
  - 🎂 Возрастная верификация — выбор возрастной категории
  - 🎮 Игровая + Пол — выбор платформы и гендера
  - 🔐 Капча — защита от ботов
  - 📚 Множественный выбор — выбор интересов (toggle)
  - 🔞 NSFW верификация — подтверждение 18+
  - 👋 Приветственная — с автоматическим ЛС
  - 🛒 Магазин ролей — выбор VIP/Премиум ролей

#### Типы действий кнопок
- `role` — выдача одной роли
- `roles` — выдача нескольких ролей (через запятую ID)
- `removerole` — удаление роли
- `togglerole` — переключение роли (есть/нет)
- `dm` — отправка ЛС пользователю
- `captcha` — текстовая капча
- `number_captcha` — математическая капча (случайные числа)
- `dropdown` — выбор роли из выпадающего списка
- `welcome` — приветственное сообщение + выдача роли
- `redirect` — перенаправление в канал
- `confirm` — подтверждение действия
- `timeout` — выдача таймаута
- `link` — кнопка-ссылка
- `kick` / `ban` — кик/бан пользователя
- `nothing` — просто кнопка без действия

#### Настройки верификации
- 🖼️ Баннер — большая картинка внизу эмбеда
- 🖼️ Миниатюра — маленькая картинка справа
- 🎨 Цвет эмбеда — HEX код
- 📝 Заголовок, описание, footer
- 📍 ID канала для верификации
- 🔘 Стиль — кнопки или dropdown

#### Команды верификации
- `/verify` — главное меню настроек
- `/verify_setup` — быстрая настройка
- `/verify_presets` — выбор готового пресета
- `/verify_preview` — предпросмотр
- `/verify_actions` — список всех типов действий

### 📂 Новые файлы
- `cogs/verifications/__init__.py` — инициализация модуля
- `cogs/verifications/verification.py` — основной код верификации

### 🗄️ Изменения в базе данных
- Добавлена таблица `verification_configs` — настройки верификации
- Добавлена таблица `verification_buttons` — кнопки верификации
- Добавлены поля в `VerificationConfig`:
  - `welcome_channel_id` — канал для приветствий
  - `banner_url` — URL баннера
  - `thumbnail_url` — URL миниатюры
  - `footer_text` — текст внизу
  - `verified_role_id` — ID верифицированной роли
  - `style` — стиль отображения

### 🔧 Исправления
- Исправлены проблемы с модальными окнами (>5 полей)
- Исправлена ошибка с StringSelect в discord.py 2.6
- Исправлены проблемы с импортами
- Добавлена поддержка загрузки cogs из подпапок
- Добавлены persistent views для verification
