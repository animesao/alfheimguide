## 🚀 Что нового?

### 🔄 Система Автообновления

Теперь бот **автоматически проверяет обновления** при запуске!

**Возможности:**
- ✅ Автоматическая проверка новых версий на GitHub
- 🔍 Сравнение версий и уведомления
- 🤖 Опциональная автоматическая установка обновлений
- 📝 Отображение заметок о релизе
- 🔒 Безопасное обновление с проверкой локальных изменений

**Как использовать:**

```bash
# Проверить обновления
python update_bot.py

# Автоматически установить обновления
python update_bot.py --auto-update
```

При запуске бота вы увидите:
```
🔍 Checking for updates...
✅ Bot is up to date!
```

Или если доступна новая версия:
```
============================================================
🎉 NEW VERSION AVAILABLE: v2026.4.17
📝 Release: Bug Fixes and Performance Improvements
🔗 URL: https://github.com/animesao/alfheimguide/releases/tag/v2026.4.17
============================================================
💡 To update, run: git pull origin main
💡 Or use: python update_bot.py --auto-update
```
