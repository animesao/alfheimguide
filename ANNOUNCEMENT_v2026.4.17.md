## 🔧 Как это работает?

### Автоматическое отслеживание
Бот проверяет релизы каждые 2 минуты для всех отслеживаемых репозиториев:

1. **Обнаружение** - Находит новые релизы
2. **Сохранение** - Записывает в базу данных
3. **Уведомление** - Отправляет красивое сообщение
4. **Фильтрация** - Не показывает черновики (drafts)

### Что отслеживается?
- ✅ Стабильные релизы
- ✅ Пре-релизы (beta, alpha, rc)
- ❌ Черновики (drafts) - не показываются

### Информация в уведомлении
- 🏷️ **Тег релиза** - версия (v2026.4.17)
- 📝 **Название** - имя релиза
- 📄 **Заметки** - описание изменений (первые 10 строк)
- 📦 **Файлы** - список для скачивания с размерами
- 👤 **Автор** - кто создал релиз
- 🌿 **Ветка** - из какой ветки
- 🔗 **Ссылки** - на страницу релиза и архивы

## 🔧 Технические детали

### Производительность
- Проверка каждые 2 минуты
- Анализ последних 5 релизов
- Кэширование в базе данных
- Оптимизированные запросы к GitHub API

### Ограничения
- Показывает до 5 файлов (assets)
- Обрезает длинные release notes (500 символов)
- Не показывает черновики
- Требует GitHub токен
